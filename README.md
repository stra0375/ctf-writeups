# ctf-writeups
Documentation of CTF challenges focusing on reasoning and methodology behind solving security challenges, demonstrating practical application of cyber security concepts.

## Areas Covered
- Web Exploitation
- Binary Analsysis
- Cryptography
- Reverse Engineering
- Forensics

## Tools & Technologies
- Programming: Python, C
- Web application security and testing: Burp Suite
- Network analysis and enumeration: Nmap, Wireshark, natcat, ssh
- Detection and threat analsysis: YARA
- Reverse engineering and binary analsysis: Objdump, strings, hexdump
- Forensics and OSINT analsysis: exiftool, steghide, zsteg, tesseract

## Repository Structure
Writeups are organised by platform (e.g. picoCTF) and category (e.g. web exploitation, cryptography) to improve navigation and clarity.

## Approach
Each writeup focuses on:
- Problem analysis and reasoning
- Tools and techniques used
- Step-by-step solution
- Key takeaways
